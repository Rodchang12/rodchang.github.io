建立Hexo部落格與使用GitHub部屬

發表於 2025-01-16 更新於 2025-01-28 分類於 技術
建立自己的Hexo 部落格吧 !

1. 關於建立部落格

在2024年開始接觸前端技術，在實體與線上課程的學習中設立了不同的目標去挑戰，並使用了 Notion 跟 HackMD 作學習筆記，經過這段時間的累積，建立自己的部落格來存放自己的筆記是更好的選項，而在眾多的平台選擇了 Hexo。

Hexo 是一個基於 Node.js 的靜態部落格框架，由於 Hexo 是使用指令來做開發，並在配合 GitHub Pages 進行部屬，可以同時練習指令與 GitHub 版本控制。

Hexo 是使用 MarkDown 撰寫格式，同時也須熟悉 Git 指令與 npm 的知識，可以參考下方連結教學。

  • MarkDown 語法大全
  • Git 教學 - 為你自己學 Git
  • npm 基本常用指令

2. 環境準備

2.1 安裝必要工具

在開始之前，請確保你的電腦已安裝以下工具：

  • Nodejs : 建議使用最新的 LTS 版本
  • Git : 版本控制
  • GitHub 帳號 : 用於存放部落格原始碼與靜態頁面
  • Visual Studio Code : 編輯器，也可以用自己習慣的 IDE 來編輯。

本文版本環境

  • 作業系統： Windows10 X64
  • Nodejs:：v20.18.2 LTS
  • IDE：VS Code

Hexo 版本

  • hexo: 7.3.0
  • hexo-cli: 4.3.2
